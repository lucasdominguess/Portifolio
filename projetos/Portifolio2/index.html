Sobre Mim

Eu sou Lucas Domingues, um jovem desenvolvedor front-end e estudante de Tecnologia da Informação. Atualmente, estou cursando Análise e Desenvolvimento de Sistemas e possuo noções básicas de tecnologias como HTML5, CSS3 e JavaScript, além de estar familiarizado com sistemas de controle de versões como Git e GitHub. Também possuo conhecimento básico na linguagem de programação Python, com foco no Back-end. Meu objetivo é aprimorar constantemente meus conhecimentos, visando me tornar um desenvolvedor de software Full-Stack no futuro. Como parte desse objetivo, criei uma página Web simples para demonstrar minhas habilidades e meu progresso no aprendizado. Espero sinceramente que apreciem!
